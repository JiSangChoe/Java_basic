현실 세계의 문제를 컴퓨터로 해결하자!!!!

프로그래밍 -> 개발

현실 세계의 정보를 데이터로 나타내기위해 코드로 표현

변수 -> 일반 변수 (하나의 표현)
배열 -> 같은 타입 데이터를 여러개 묶어 표현
구조체 -> 서로 다른 타입의 데이터를 묶어서 표현
클래스 -> 기능을 묶어서 표현

JVM, JRE, JDK

JVM -
메서드 영역: 클래스나 인터페이스의 구조 정보, 정적 데이터(static)가 저장되는 메모리 공간
힙 영역: 인스턴스의 데이터(new 연산자로 생성된 데이터)가 저장되는 메모리 공간
스택 영역: 하나의 스레드 (호출된 메서드)에서 사용되는 변수에 대한 데이터가 저장되는 메모리 공간
